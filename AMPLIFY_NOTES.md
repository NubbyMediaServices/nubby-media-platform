# Amplify Notes

This project contains two Amplify-hosted frontends.

## App 1: Website

Amplify app root:

```text
website/
```

Build settings file:

```text
website/amplify.yml
```

Output directory:

```text
dist
```

## App 2: Client Portal

Amplify app root:

```text
client-portal/
```

Build settings file:

```text
client-portal/amplify.yml
```

Output directory:

```text
dist
```

## Backend

The backend is not hosted by Amplify static hosting. Deploy it separately and expose it as:

```text
https://api.nubbymedia.com
```
